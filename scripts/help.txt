!гена -- случайное предложение с единичной связью в словах
!генаа -- случайное предложение с двойной связью в словах
!гена(word,min,max,mode) -- параметры необязтаельные, можно использовать только часть из них, но слева направо, т.е. можно использовать только word, но нельзя использовать только max
word -- слово, min/max -- минимальное/максимальное количество слов в предложеении, mode -- режим 0 или 1,если 1, то он ищет только в тех предложениях, где word является началом предложения
